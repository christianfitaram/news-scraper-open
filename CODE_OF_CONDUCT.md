# Code of Conduct

## Our Standard

Participation in this project should be constructive, respectful, and professional.

Examples of behavior that contribute to a positive environment:

- giving clear, respectful feedback
- assuming good intent
- focusing criticism on code and behavior, not people
- documenting tradeoffs and decisions clearly

Examples of unacceptable behavior:

- harassment or personal attacks
- discriminatory or abusive language
- repeated bad-faith arguments
- publishing private information without permission

## Enforcement

Project maintainers may remove comments, reject contributions, or block participation that violates this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, report it privately to the maintainer.
